# SecureCodeAI

SecureCodeAI is a security analysis tool that helps developers identify common vulnerabilities in source code.

## Features

- Hardcoded Password Detection
- API Key Detection
- Security Score Calculation
- Risk Level Analysis
- Security Recommendations
- User Friendly Interface

## Technologies Used

- HTML
- CSS
- JavaScript
- Java

## How to Run

1. Open the project in VS Code.
2. Open frontend/index.html.
3. Run using Live Server.
4. Paste code into the scanner.
5. Click Scan Code.
